# Git Cheatsheet

## Repository maken
git init

## Bestand toevoegen
git add .

## Commit maken
git commit -m "bericht"

## Remote toevoegen
git remote add origin <repo-url>

## Push naar GitHub
git push -u origin main

## Branch maken
git branch nieuwe-branch

## Wisselen van branch
git checkout nieuwe-branch

## Branch pushen
git push origin nieuwe-branch

## extra tips
git status
